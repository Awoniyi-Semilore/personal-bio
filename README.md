*🚀 Project Overview*

This repository contains a premium, high-impact Personal Bio Page designed with a "Bold & Unrestrained" aesthetic. The goal was to move beyond basic HTML/CSS to create a professional-grade digital presence that prioritizes clean typography, accessibility, and modern UI/UX principles.


*🛠 Technical Specifications*

 * Architecture: Single-file HTML5/CSS3 implementation.
 * Typography: Integrated Google Fonts (Space Grotesk & Inter) for a high-end feel.
 * Form Design: Fully semantic <form> structure using <label> elements linked via for and id for maximum accessibility.
 * Styling: - Glassmorphism effects using backdrop-filter.
   * Responsive design optimized for mobile (Acode environment) and desktop.
   * Custom CSS variables for centralized theme management.

   
*📱 Mobile Development Workflow*

This project was developed entirely on a mobile device using:

 * Editor: Acode (Android/iOS)
   
 * Version Control: GitHub Web Interface
   
 * Preview: Mobile Chrome / Acode Previewer
